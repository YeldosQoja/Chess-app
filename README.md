## Installation

To get a local copy up and running, follow these simple steps:

1. **Clone the repo**
   ```sh
   git clone https://github.com/YeldosQoja/chess-game-app.git
2. **Install dependencies**

   ```bash
   npm install
   ```
2. **Start the app**

   ```bash
    npx expo start
   ```
